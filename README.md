# iskillsbiz – Chatbot Broadcasting Platform

## Project Overview
This is a **conversion-focused SaaS landing page** for **iskillsbiz**, a chatbot broadcasting service.

The platform allows businesses to broadcast messages to thousands of subscribers across:
- Facebook Messenger
- Telegram
- Instagram
- Future chat platforms

This project is a **static landing page only**.
No backend, no database, no authentication.

Language: English only  
Style: Modern SaaS, professional, clean, high-conversion
